# 多情PC28 · 实战分析

一个基于 HTML5 的 PC28 彩票分析工具，提供实时数据分析、三步算法预测、走势图展示等功能。

## 功能特性

- **实时数据获取**：从 API 获取最新的 PC28 开奖数据
- **三步算法**：基于历史数据的预测算法
- **走势图分析**：可视化展示最近30期数据走势
- **冷热分布**：统计各种形态的出现频率
- **推荐与杀号**：基于数据分析的推荐和杀号建议
- **离线模式**：当 API 无法访问时，使用示例数据展示功能

## 部署到 Vercel

本项目已配置 `vercel.json`，可直接部署到 Vercel。

### 快速开始

1. 克隆或创建 GitHub 仓库
2. 连接到 Vercel
3. 自动部署完成
